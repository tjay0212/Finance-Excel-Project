# Apple Inc. — Financial KPI Dashboard

A dynamic Excel dashboard analysing Apple's financial performance FY2019–2023.

## What's included
- Revenue, Net Income, Gross Margin, Net Margin KPI cards
- Dynamic year selector (FY2019–FY2023)
- Revenue trend, margin trend, and YoY growth charts
- Data cleaning exercise (Raw → Clean Data pipeline)

## Key findings
- Revenue grew at a 10.1% CAGR from $260bn to $383bn
- Gross margin expanded from 37.8% to 44.1% over the period
- Net income peaked at $99.8bn in FY2022 before a slight pullback
- Apple repurchased over 1,000M shares across the 5-year period

## Tools used
Excel — PivotTables, XLOOKUP, Data Validation, dynamic charts
