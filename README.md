# Apple Inc. — Financial KPI Dashboard
A dynamic Excel dashboard analyzing Apple's financial performance fromm FY2019 to FY2023, built to demonstrate skills in fiancial analysis, Excel automation and dashboard design.

## Project Overview
This project transforms Apple's 5-year financial satmenets into an interactive KPI dashboard. It highlights revenue grwoth, margin trends, and profitability using a clean analyst style layout. 

## What's included
- Revenue, Net Income, Gross Margin, Net Margin KPI cards
- Dynamic year selector (FY2019–FY2023)
- Revenue trend, margin trend, and YoY growth charts
- A full Raw -> Clean Data pipeline

## Key findings
- Revenue grew at a 10.1% CAGR from $260bn to $383bn
- Gross margin expanded from 37.8% to 44.1% over the period
- Net income peaked at $99.8bn in FY2022 before a slight pullback
- Apple repurchased over 1,000M shares across the 5-year period

## Tools used
- Excel — PivotTables, XLOOKUP, Data Validation, dynamic charts
- Financial Analysis - CAGR, margin analysis, YoY growth
- Data Cleaning - structured references, formula-driven practices
- Dashboard Design - KPI cards, interactive selectors, layout best practices
- Financial KPIs - revenue, margins, profitability metrics

## How to Use the Dashboard
- Open Apple_KPI_Dashboard_Excel.xlsx
- Use the year dropdown to switch between FY2019–FY2023
- Explore:
- KPI cards
- Revenue trend
- Margin trend
- YoY growth chart

## Methodology
Data Cleaning
- Standarized fiscal years
- Normalized revenue, cost, and margin fields
- Built a structured "Clean Data" table
- Applied XLOOKUP for dynamic  KPI retrieval

KPI Calculations
- Revenue CAGR
- Gross Margin %
- Net Margin %
- YoY growth Formulas
